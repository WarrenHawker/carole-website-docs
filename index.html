Introduction

All the artwork for the Carole Tongue website is managed by a system called "Contentful". This gives you a place to add, edit, unpublish and delete artwork with no coding required. In this guide we will go through all the different things you can do on "Contentful" and how it affects how the artwork looks on the main website.

Login to Contentful

To login, go to https://be.contentful.com/login and login with your credentials.

If you forget your password, click the "Reset password" button and enter your registered email address when prompted. You will receive an email with a password reset option to recover your account. If this doesn't work, contact the administrator.

Once logged in you will be automatically taken to the Content page where you can see all the published artwork.

Add new media

The images for any new pieces of artwork need to be added as media before they can appear on the main website. This can either be done as you create the new artwork entries (see below) or ahead of time by adding media. To add new media, go to the Media tab at the top on the screen. You should now see all the images that have currently been uploaded.

To add new media, click the Add Asset button, then select either "single asset" to upload one image at a time, or "multiple assets" to upload multiple images at once.

Adding single media

Adding a single media piece lets you give it a specific title and description. These are optional and won't be displayed on the main website (they are for your reference only). If you choose not to give a title, it will use the same title as the file name.

Inside the "File(required)" box, click the Open file selector button. From here you can select an image from a number of different sources, including your PC, Facebook, Instagram, etc.. Or drag'n'drop images directly into the window

Once you've chosen your image, a preview of it will appear in the box. Below the box there are a number of options for altering the image if you aren't happy with it. If you wish to use a different image, click the "delete file" button and choose a different file as before.

Once you are happy with your image, click the green Publish button on the right hand side to save your work.

Adding multiple media

When adding multiple media pieces, the titles will be the same as the file names. Inside the popup box, select the media pieces from your chosen source, or drag'n'drop them directly into the window.

Clicking on the "edit" button next to an image will allow you to edit the image by cropping or rotating it. To remove an image from the selection, click the "X" button next to the image you want to remove. Once you're happy with all the images, click the Upload button to save your work.

NOTE: Images of artwork only show up on the main website when they are linked to an Artwork entry. See below for how to do this.

Add new artwork

To get started, go to the Content tab at the top of the screen and you should see all the published artwork.

To add a new artwork entry, click the Add Entry button on the right hand side.

On the following page, provide a Title, Description (optional), image, created on(date) and category. These settings will define how the artwork is displayed on the main website.

NOTES:

  • The Description will only appear on the page for the individual artwork on the main website. If you don't provide a Description for the artwork, this section will be blank.
  • To add an image, click the Add media button and then either select "Add existing media" or "Add new media".
    • Clicking "Add existing media" will allow you to select any media you have previously uploaded through the "Media" tab or any other published work.
    • Clicking "Add new media" will bring up the window for uploading a new media asset (see Add new media for details).
    • If you want to change the image after selecting it, click on the 3 little dots in the box and either select "edit" to alter the image or "remove" to remove it and select a different image.
  • The created on(date) will show up underneath the artwork title on the main website, and will only display the month and year.
  • Any category you select will cause the artwork piece to show up in that category section on the main website. If you select multiple categories your artwork will show up in those multiple categories on the main website.
  • The slug is the web address for the individual page for that specific artwork, and is based on the title of the piece. For example if the slug is "Joy", the address for that individual piece will be https://carole-website-contentful.vercel.app/art/joy
    • Note: you cannot have 2 artwork pieces with the same slug
  • The final setting is front page image. Selecting "yes" here will cause the image to show up on the front page of the main website. However, if you have selected "yes" on multiple pieces of artwork, only ONE of them will appear on the front page.

Once you are happy with your settings, click the green Publish button on the right hand side to publish the artwork to the main website.

Edit artwork

To get started, go to the Content tab at the top of the screen and you should see all the published artwork.

To edit a piece of artwork after publishing it, click the name of the artwork you wish to edit.

Make the changes as you see fit (remember that all "required" fields must have something in them).

Once you are happy with your edits, click the green Publish Changes button on the right hand side to publish the artwork to the main website.

Unpublish artwork

Unpublishing an artwork piece will remove it from the main website but not delete it entirely. To get started, go to the Content tab at the top of the screen and you should see all the published artwork.

Click on the little white square next to the title of the artwork piece you wish to unpublish.

Click the Unpublish button. Once done, the artwork piece should have a "draft" status and will no longer show up on the main website.

To re-publish an artwork piece again, Click on the little white square next to the title and select the green Publish

Delete artwork

Deleting an artwork piece will delete everything about it - This process is irreversible. To get started, go to the Content tab at the top of the screen and you should see all the published artwork.

You can only delete unpublished artwork, so do that first (see previous section).

Once unpublished, Click on the little white square next to the title, then click the red Delete button.
